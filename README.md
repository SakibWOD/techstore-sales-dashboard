# 📊 TechStore Sales Dashboard

## Project Overview

This project analyzes sales data using SQL and visualizes insights using an Excel dashboard.

##  Tools Used

* Microsoft Excel (Dashboard & Visualization)
* MySQL (Data Analysis)

##  Key Insights

* Dhaka generates the highest revenue, indicating a strong premium market.
* Electronics contributes the highest share of total sales.
* Sales peak in January and decline afterward.
* Stationery has the lowest contribution.

##  Files Included

* Excel dataset and dashboard
* SQL queries for analysis
* Dashboard screenshot

##  Skills Demonstrated

* Data Cleaning
* SQL (JOIN, GROUP BY, CASE WHEN)
* Dashboard Design
* Business Insight Generation
